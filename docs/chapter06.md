# 第6章：重要経典10選 - 読むべき箇所ガイド

読者が自らパーリ経典に触れるための道標となる章である。2万ページの経典から、現代人にとって最も実践的で理解しやすい経典を厳選し、それぞれの特徴と読み方のコツを提供する。

重要なのは、経典を学術的に研究することではなく、日常生活で実際に活用することである。各経典の「使い方」に焦点を当てて解説する。

これらの経典を通じて、読者は2500年前の智慧を現代生活で実際に活用することができる。重要なのは、学術的な知識の蓄積ではなく、日常的な実践を通じた人生の質の向上である。各経典の教えを少しずつでも実際に試してみることで、テーラワーダ仏教の真の価値を体験することができるだろう。
